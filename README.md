## Описание:
В этом приложении использованы react хуки, такие как: useContext, createContext, useReducer, useEffect

### Установка и запуск

```bash
# Устанавливаем зависимости
npm install

# Запускаем приложение для разработки.
npm run start

# Запускаем приложение в production-режиме.
npm run build
```
